#  Hospital Management System

##  Description
This is a simple Hospital Management System developed using Python and Tkinter. It helps manage patients, doctors, appointments, and billing.

##  Features
- Add, Update, Delete Patients
- Add, Update, Delete Doctors
- Book Appointments
- Generate Bills
- Display Data in Table

##  Technologies Used
- Python
- Tkinter (GUI)

##  How to Run
1. Download the project
2. Run main.py
